Question-1: What are the different ways to select an element in the DOM?

getElementById, getElementsByClassName, QuerySelector and QuerySelectorAll can be used to select an element in the DOM

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML means HTML code with tags, innerText means text inside an element and textContent is a node's contents

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where you attach a single event listener to a parent element to manage events on its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling in JavaScript refers to the way events are handled and propagated through the different elements of a webpage.

Question-5: How do you create, add, and remove elements using JavaScript?

I can create element with createElement, add with appendChild & remove with remove() or removeChild()
